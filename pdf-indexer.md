# pdf-indexer

Use para ingestao e busca semantica em PDFs.

- O MCP deve ser instalado a partir do repositorio oficial, nao por dump local.
- Leia o README de `https://github.com/lizTheDeveloper/pdf-indexer-mcp` e siga as instrucoes de instalacao.

Fluxo tipico:

1. `download_pdf`
2. `index_pdf`
3. `generate_embeddings`
4. `search_research_papers`
