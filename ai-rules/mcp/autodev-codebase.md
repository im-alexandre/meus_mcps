# autodev-codebase

Use este MCP como primeiro passo para exploracao do repositorio.

- Use `outline_codebase` para mapear modulos, classes e funcoes sem ler arquivos inteiros.
- Use `search_codebase` para perguntas em linguagem natural sobre comportamento do codigo.
- Recorra a `rg`, `find`, `Grep`, `Glob` ou leitura direta apenas quando houver divergencia entre o indice e o estado real.
- Se houver divergencia, reindexe antes de continuar.

Instalacao:

- leia o README de `https://github.com/anrgct/autodev-codebase`
- siga as instrucoes oficiais do repositorio
